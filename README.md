# 🏋️ Gym Manager

## 📌 Status do Projeto
🚧 Em desenvolvimento

---

## 💻 Tecnologias utilizadas
- Java
- Java Swing (Interface gráfica)
- MySQL
- JDBC (conexão com banco de dados)

---

## 👨‍💻 Desenvolvedor

---

## 🎯 Objetivo do Software
O sistema Gym Manager tem como objetivo auxiliar no gerenciamento de academias, permitindo o controle de alunos, planos e treinos de forma simples e organizada.

---

## ⚙️ Funcionalidades do Sistema

### 👤 Alunos
- Cadastro de alunos
- Listagem de alunos
- Associação com planos

### 🏋️ Treinos
- Cadastro de treinos
- Associação de treino com aluno
- Registro de data do treino

### 📊 Dashboard
- Total de alunos cadastrados
- Quantidade de planos ativos
- Treinos realizados no dia

---

## 📌 Observações
O sistema ainda está em desenvolvimento e novas funcionalidades serão adicionadas futuramente, como edição e exclusão de dados, melhorias na interface e relatórios avançados.
